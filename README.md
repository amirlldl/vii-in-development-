# vii-in-development-
text editor created on pascal, still on development and must be added base functionality,that it havent,like navigate between strings and reading old files
to create new text in file follow this steps:
1.compile navigate(if you dont done that) with fpc testnav1.pas
2.create new text file with ./testnav1 your_file
3.write something in it(or dont write - you are on your own)
-to enter insert mode enter the 'i' symbol
-to exit from insert mode enter 'esc' button
-to quit from the text editor type 'q'
4.File will save automaticly when you type 'q'
